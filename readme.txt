# Course Report App

Foobar is a Python library for dealing with word pluralization.

## Installation

Use npm package manager

```bash
npm install
```
Use nodemon

```bash
npm install nodemon
```

## Starting Proxy Servers

Start up the rails server:
```bash
rails s
```
Start up the node server:
```bash
nodemon server.js
```
Start up the expo server:
```bash
expo start
```
